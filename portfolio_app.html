Portfolio App (Work in progress)

This is a dockerized Rest Api build with Rest Framework.

I am using celery tasks to get and store the assets from the CoinmarketCap API

and also to update the prices.

The user is able to create multiple portfolios and add assets to them.

The next step is to aggregate the data and display the summary and statistics.(coming soon)

The Frontend part is build by another developer.
